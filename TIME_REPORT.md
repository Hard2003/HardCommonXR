# Time Report - TCXR Cares Takehome Assignment

**Total Time:** 2.5 hours  
**Date:** December 31, 2025  
**Status:** All 5 tasks completed successfully  

## Task Breakdown

**Task 1 - Student List Display** took 25 minutes to implement the basic table structure and styling.

**Task 2 - Institution List Display** required 20 minutes, building on the previous task's foundation.

**Task 3 - Data Caching Implementation** was the most challenging, taking 45 minutes to properly implement React Context API for secure data management.

**Task 4 - Institution Student Roster** took 35 minutes, including backend URL parameter handling and data filtering.

**Task 5 - Dashboard Creation** required 45 minutes to design meaningful visualizations with Plotly.js for both administrator and teacher roles.

## Key Challenges

The biggest challenge was implementing React Context API for data caching, which required learning advanced state management patterns. URL parameter handling in the Python backend also presented difficulties with encoding and decoding institution names. Creating meaningful educational visualizations required careful consideration of what administrators and teachers actually need to see.

**AI Usage:** Used ChatGPT briefly for React Context syntax guidance and Plotly.js chart examples, as these were new technologies for me.

## Learning Outcomes

This project provided valuable experience with advanced React state management through the Context API. I gained practical knowledge of backend-frontend integration and learned to create effective data visualizations. The assignment also improved my understanding of professional full-stack development practices and user-centered design thinking.
